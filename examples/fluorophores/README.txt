Some of the flourescence scenes use data from external websites.
Due to copyright reasons, they are not included in source and the user has to download it to render the scenes themselves.

All the data is acquired here:
https://www.thermofisher.com/de/de/home/life-science/cell-analysis/labeling-chemistry/fluorescence-spectraviewer.html

Necessary fluorophores are:

 - MarinaBlue.csv (Marina Blue)
 - HypoxiaGreen.csv (Hypoxia Green)
